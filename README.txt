Программа расчета налогового вычета на лечение для физических лиц

Веб-сайт поддерживается такими бразерами как Yandex, Google, Mozilla

Для Windows:
1. Перейдите в один из перечисленных браузеров
2. Загрузите все файлы проекта в одну свежесозданную директорию и откройте файл index.html при помощи браузера.

Для Linux:
Зайдите в барузер Mozilla и загрузите все файлы проекта в одну свежесозданную директорию и откройте файл index.html при помощи браузера.

Для MacOS:
Техники на сиситеме MacOS в наличии не было, не мог проверить в браузере Safari, 
поэтому нужно будет установить один из перечисленных браузеров!

1. Перейдите в один из перечисленных браузеров
2. Загрузите все файлы проекта в одну свежесозданную директорию и откройте файл index.html при помощи браузера.
